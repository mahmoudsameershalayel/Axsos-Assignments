# MySQL Countries Queries

This project contains SQL query practice for a MySQL countries database.

## Contents

- `MySQL_Countries.txt` - answers to country, city, and language query questions.

## Database Tables Used

The queries use these tables:

- `countries`
- `cities`
- `languages`

The examples practice common SQL skills, including:

- `JOIN`
- `WHERE` filtering
- `GROUP BY`
- `COUNT`
- `ORDER BY`
- subqueries

## How To Use

Open `MySQL_Countries.txt`, copy a query, and run it in MySQL Workbench or another MySQL client connected to the countries database.

Each numbered section includes the question followed by the SQL query answer.

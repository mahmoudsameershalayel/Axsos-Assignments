# HTML Table with Flask

A simple Flask application that displays a list of users in an HTML table.

## Project Structure

```
HTML-table/
├── main.py              # Flask application
├── templates/
│   └── index.html       # HTML template with user table
└── README.md           # This file
```

## Features

- Displays a list of users in a responsive HTML table
- Uses Foundation CSS framework for styling
- Features hover effects on table rows
- Clean and modern UI design

## Requirements

- Python 3.x
- Flask

## Installation

1. Install dependencies:
   ```bash
   pip install flask
   ```

2. Run the application:
   ```bash
   python main.py
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:5000
   ```

## Usage

The application displays a table with user information (first name and last name). The sample data includes four users that are rendered in the HTML template.

## Technologies Used

- Flask (Web Framework)
- HTML/CSS
- Foundation CSS Framework

# Python Dictionaries & Lists Practice

## Description

This project demonstrates basic operations in Python using lists and dictionaries. It focuses on updating values, iterating through data structures, and extracting specific information.

---

## Features

* Updating values inside nested lists and dictionaries
* Modifying dictionary values inside lists
* Iterating through a list of dictionaries
* Extracting specific keys from multiple dictionaries
* Working with dictionaries that contain list values

---

## What I learned

* How to access and modify nested data structures
* How loops work with dictionaries and lists
* How to structure and print formatted output from complex data

---

## Purpose

This exercise helps strengthen understanding of Python data structures and improves problem-solving with real-world data patterns.

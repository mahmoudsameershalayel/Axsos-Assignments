# Zoo OOP

This is a simple Python object-oriented programming exercise that models a zoo with different animals.

## Files

- `Animal.py` - Base `Animal` class with shared attributes and methods.
- `Lion.py` - `Lion` class that inherits from `Animal`.
- `Bear.py` - `Bear` class that inherits from `Animal`.
- `Monkey.py` - `Monkey` class that inherits from `Animal`.
- `Zoo.py` - `Zoo` class that stores animals and manages feeding/displaying them.
- `main.py` - Creates a zoo, adds animals, feeds them, and prints their info.

## Run

```bash
python main.py
```

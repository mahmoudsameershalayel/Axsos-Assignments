# Bank Account Assignment (Python OOP)

## Description

This project simulates a basic banking system using Python classes. It supports deposits, withdrawals, interest calculation, and method chaining for cleaner code execution.

---

## Features

* Create bank accounts with interest rate and balance
* Deposit money into the account
* Withdraw money with validation
* Apply withdrawal fee if balance is insufficient
* Calculate and add interest to balance
* Display account information
* Support method chaining for multiple operations in one line

---

## Concepts Used

* Object-Oriented Programming (OOP)
* Method chaining
* Condition handling
* Encapsulation of financial logic
* Basic interest calculation

---

## How it works

* Each account is created with an initial balance and interest rate
* Users can perform multiple actions in a single chained statement
* Withdrawals check balance before processing
* Interest is added only if balance is positive
* Account info can be displayed anytime

---

## Purpose

This project helps practice OOP design patterns in Python, especially method chaining and real-world financial logic simulation.

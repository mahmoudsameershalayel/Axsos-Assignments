# User Bank System (Python OOP)

## Description

This project simulates a simple banking system using Python classes. Each user has a name, email, and account balance, and can perform basic banking operations.

---

## Features

* Create user accounts with initial balance
* Deposit money into account
* Withdraw money with balance validation
* Transfer money between users
* Display current account balance

---

## Concepts Used

* Object-Oriented Programming (OOP)
* Classes and Objects
* Instance methods
* Condition checking (validation logic)
* Interaction between objects

---

## How it works

* Each user starts with a balance of 0
* Users can deposit and withdraw money
* Withdrawals are blocked if funds are insufficient
* Money can be transferred between users safely
* Each action updates the user’s balance

---

## Purpose

This project helps practice real-world OOP concepts like encapsulation, object interaction, and method design in Python.

# Likes MySQL Database Schema

A MySQL Workbench database model for a likes/reactions system, built as part of the Axsos Academy SQL assignments.

## File

- `Likes.mwb` — MySQL Workbench model file containing the database schema

## Usage

Open `Likes.mwb` with [MySQL Workbench](https://www.mysql.com/products/workbench/) to view the EER diagram and schema structure.

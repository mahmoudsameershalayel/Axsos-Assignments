# Food Reviews Database

A MySQL database schema designed for managing food reviews, built with MySQL Workbench.

## Files

- `Food_Reviews.mwb` — MySQL Workbench model file containing the database schema

## How to Open

1. Open **MySQL Workbench**
2. Go to **File > Open Model**
3. Select `Food_Reviews.mwb`

## How to Generate the Schema

1. Open the model in MySQL Workbench
2. Go to **Database > Forward Engineer**
3. Follow the wizard to generate and run the SQL on your MySQL server

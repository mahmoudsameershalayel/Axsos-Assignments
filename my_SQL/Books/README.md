# Books MySQL Database Schema

A MySQL Workbench database model for a books management system, built as part of the Axsos Academy SQL assignments.

## File

- `Books.mwb` — MySQL Workbench model file containing the database schema

## Usage

Open `Books.mwb` with [MySQL Workbench](https://www.mysql.com/products/workbench/) to view the EER diagram and schema structure.

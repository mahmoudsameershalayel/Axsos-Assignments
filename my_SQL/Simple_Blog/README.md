# Simple Blog MySQL Database Schema

A MySQL Workbench database model for a simple blog platform, built as part of the Axsos Academy SQL assignments.

## File

- `Simple_Blog.mwb` — MySQL Workbench model file containing the database schema

## Usage

Open `Simple_Blog.mwb` with [MySQL Workbench](https://www.mysql.com/products/workbench/) to view the EER diagram and schema structure.

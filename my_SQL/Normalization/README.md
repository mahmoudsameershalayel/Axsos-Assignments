# Normalization

This folder contains a MySQL Workbench model for a database normalization assignment.

## Files

- `Normalization.mwb` - the main MySQL Workbench model file.
- `Normalization.mwb.bak` - a backup copy of the model.

## How to Open

1. Open MySQL Workbench.
2. Choose **File > Open Model**.
3. Select `Normalization.mwb`.

## Purpose

The project is used to practice database normalization, including organizing data into related tables and reducing duplicate or inconsistent data.

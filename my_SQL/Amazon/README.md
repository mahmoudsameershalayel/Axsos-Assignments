# Amazon MySQL Database Schema

A MySQL Workbench database model for an Amazon-like e-commerce system, built as part of the Axsos Academy SQL assignments.

## File

- `Amazon.mwb` — MySQL Workbench model file containing the database schema

## Usage

Open `Amazon.mwb` with [MySQL Workbench](https://www.mysql.com/products/workbench/) to view the EER diagram and schema structure.

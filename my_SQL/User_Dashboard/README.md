# User Dashboard

This project contains a MySQL Workbench database model for a User Dashboard assignment.

## Files

- `User_Dashboard.mwb` - the main MySQL Workbench model file.
- `User_Dashboard.mwb.bak` - a backup copy of the model file.

## How to Open

1. Open MySQL Workbench.
2. Go to `File > Open Model`.
3. Select `User_Dashboard.mwb`.

## Purpose

The database model is intended to represent the structure needed for a user dashboard system. It can be used to review the schema, edit tables and relationships, and generate SQL scripts from MySQL Workbench.

## Exporting SQL

To export the database schema:

1. Open the model in MySQL Workbench.
2. Go to `Database > Forward Engineer`.
3. Follow the steps to generate and export the SQL script.

# Blogs MySQL Database Schema

A MySQL Workbench database model for a blogging platform, built as part of the Axsos Academy SQL assignments.

## File

- `Blogs.mwb` — MySQL Workbench model file containing the database schema

## Usage

Open `Blogs.mwb` with [MySQL Workbench](https://www.mysql.com/products/workbench/) to view the EER diagram and schema structure.
